# Life & Space Tracker

A web app that transforms your birth date into meaningful life statistics and cosmic insights.

## Features

- Exact age (years, months, days)
- Total days lived
- Countdown to next birthday
- Zodiac sign
- Estimated heartbeats
- Distance traveled around the Sun
- Moon cycles experienced
- Distance traveled through the Milky Way
- Total sleep time (approximate)
- Dark mode toggle
- Saves your data using localStorage

## Tech Stack

- HTML5
- CSS3 (with gradients & dark mode)
- Vanilla JavaScript

---

## How It Works

The app calculates:
- Time differences from your birth date
- Astronomical approximations (Earth orbit, galaxy movement)
- Biological estimates (heartbeats, sleep)

All calculations are done in real-time using JavaScript.

---

## Disclaimer

All values are approximate and for educational or entertainment purposes only.
